What Net Ward sells

Open-source software with voluntary sponsorship.

Net Ward is a user-space reverse proxy that helps small HTTP services deflect automated probes and low-end abuse with safe HTTP mirror responses. Payments through sponsorship support software development and maintenance; they are not charitable donations, tax-deductible gifts, managed security services, or peer-to-peer money transfer.

Included channels

GitHub issues, public documentation, security policy, changelog, and repository discussions through normal GitHub workflows.

Alpha boundaries

No guaranteed response time, no managed hosting, no emergency incident response, and no lifetime custom support promise.

Security scope

Net Ward uses harmless HTTP responses. It does not attack visitors, distribute malware, or retaliate against bots.
